This project:

1. Collects Amnesty Urgent Actions and metadata
2. Cleans them as a csv
3. Pulls out the names of individuals mentioned in UAs.
4. Runs those names against the New York Times article API
5. Creates a database of 'mentions' of Amnesty UAs in New York Times (from steps 2,3,4 above)
6. Analyzes that database for patterns.
